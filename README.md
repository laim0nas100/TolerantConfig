# TolerantConfig
Java extremely simple and performant multi-level config, based on Properties or any Map.
Tolerates exceptions, missing or misconfigured properties if you supply a default value.
Can optionally throw excpetions on missconfigured or missmapped properties.
Supports property indirection for static typed properties that can change values based on provided configs or given environments.
Supports arbitrary property value mapping.
No interpolation, working purely on the provided Map instance - no hidden behavior.
